scidb4gdal - A GDAL driver for scidb arrays

This driver interfaces SciDB for accessing two-dimensional arrays. Detailed instructions can be found at https://github.com/mappl/scidb4gdal.

Connection String

Syntax: 'SCIDB:array=%arrayname%[ host=%hostname%][ port=%portnumber%][ user=%username%][ password=%password%]

where:

  • array is the name of a SciDB array
  • host is an URL to connect to shim (default http://localhost)
  • port is the port shims listens on (default 8080)
  • user is the user name for shim authentication (default scidb)
  • password is the password for shim authentication (default scidb)

Creation issues

Currently the driver is read only.

Overviews

The driver does not support building overviews.

Examples:

  • Accessing a rasterlite DB with a single raster table :
