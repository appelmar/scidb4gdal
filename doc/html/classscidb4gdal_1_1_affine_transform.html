Detailed Description

Class to store transformation parameter and to perform transformations with that.

Affine transformations are used to map array integer coordinates to world SRS coordinates. This class stores six parameters of a 2D affine transformation and provides basic serialization functions.

See Also
GDAL data model specification http://www.gdal.org/gdal_datamodel.html
